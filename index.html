    Inline & Block Elements

    Inline Elements

    Inline elements doesn't start on new line and takes only necessary width of container for content

    Example of Inline tags:a,span,img,video,audio,iframe,input,label,b,i,em,strong,sub,sup etc

    Block Elements

    Block elements start on new line and pushes next element to new line

    Takes full width of container.Width and height can be controlled using CSS

    Can contain other block and inline elements inside it

    Examples for block elements:p,div,header,footer,ul,ol,li,dl,dt,dd,form,table etc

    We can use CSS property 'display' to make inline element to block element and vice versa

    Id and Classes

    Id and class,both are attribute which is used to identify elements while there is some difference between them ids are unique identifier assigned to a single element on page.Multiple elements can't have same id.It can be used for unique styling of an element or JavaScript manipulation for that element only

    Multiple elements can have same class name and same element can have multiple class names.They are not unique.They can be used to change look and behaviour of all elements with same class name at once 

        Example:
    
      <div id="firstDiv" class="divs entryDiv">Hello</div>
      <div id="secondDiv" class="divs exitDiv">Bye</div>
